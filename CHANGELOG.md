# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2024-11-26

### Added
- Initial release of vue-flip-clock component
- Clock mode for displaying current time
- Countdown mode for counting down to a target time
- Timer mode for counting up from a start time
- Smooth 3D flip animations
- Vue 3.5 support with Composition API
- Modern build setup with Vite 7
- Comprehensive documentation and examples

### Features
- Three display modes: clock, countdown, and timer
- Customizable via SCSS
- TypeScript support via JSDoc
- Lightweight with minimal dependencies
- Optimized performance with modern Vue 3 patterns
